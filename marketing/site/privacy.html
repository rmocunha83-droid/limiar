Política de privacidade

Limiar foi desenhado para criar uma pausa espiritual antes de apps que roubam atenção. Esta política explica quais dados o app usa e como eles são tratados.

Dados armazenados no dispositivo

O app pode salvar localmente suas preferências de leitura, tradição espiritual escolhida, temas favoritos, histórico de leituras, passagens favoritas e configurações das pausas. Esses dados ficam no dispositivo e são usados para operar a experiência do app.

Permissões do Tempo de Uso

Quando você escolhe apps, categorias ou sites que vão ativar o Limiar, o app usa recursos nativos do iOS, como FamilyControls e DeviceActivity. A seleção é usada para criar pausas no próprio dispositivo. O Limiar não vende, compartilha ou usa essa seleção para publicidade.

Reflexões e conteúdo espiritual

O app exibe trechos, resumos e reflexões para meditação pessoal. Esses conteúdos não substituem orientação religiosa, pastoral, rabínica, psicológica, médica ou profissional.

Geração por IA

Para gerar reflexões personalizadas, o Limiar pode enviar ao backend apenas os dados necessários: tradição espiritual escolhida, temas, livros ou seções preferidas, profundidade da explicação, trecho selecionado e um histórico recente resumido de reflexões para evitar repetição. O backend usa uma chave segura para chamar um provedor de IA generativa. O app não envia identificadores pessoais, seleção dos apps que ativam o Limiar, email, contatos, localização precisa ou dados de publicidade para essa geração.

No Modo Essencial, o app mostra apenas trechos principais e não envia chamadas ao backend para gerar reflexões por IA ou narração.

Rastreamento e publicidade

O Limiar não usa rastreamento publicitário, não vende dados pessoais e não compartilha dados com redes de anúncios.

Suporte

Se você entrar em contato para suporte, poderemos receber as informações que você decidir enviar, como email, descrição do problema, modelo do dispositivo ou prints. Esses dados são usados apenas para responder ao atendimento.

Retenção e controle

Preferências e histórico podem ser removidos ao apagar o app do dispositivo. Alguns dados também podem ser alterados dentro do próprio app, quando a interface oferecer essa opção.

Alterações

Se as práticas de privacidade mudarem, esta página será atualizada. A data no topo indicará a versão mais recente.

Contato

Para perguntas sobre privacidade, escreva para suporte@limiar.app.
